# @spool-ai/shared

Part of [Spool](https://github.com/HoneycombHairDevelopers/Spool) — the debugger for AI agents. Capture every run, inspect every decision, pause and inject live, fork from any step.

Shared domain types (Run, Step, Action, Outcome, FileChange), SHA-256 hashing, secret redaction, and the Live Probe file protocol.

```bash
npm install @spool-ai/shared
```

See the [Spool documentation](https://github.com/HoneycombHairDevelopers/Spool#readme) for the full guide. MIT licensed.
